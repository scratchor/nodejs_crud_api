### Hello! This is Node.js CRUD API

- ### Routes
      GET      /api/users
      POST     /api/users
      GET      /api/users/id
      PUT      /api/users/id
      DELETE   /api/users/id

- ### Example :
      GET http://localhost:3000/api/users- to get all users

- ###Install
      npm install

- ###Development mode
      npm run dev

- ###Production mode
      npm run build
